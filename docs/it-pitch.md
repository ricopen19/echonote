---

## 1. 概要

**Echonote** は、音声ファイルを文字起こし・要約し、議事録や記録ドキュメントを自動生成するアプリケーションです。会議・研修・授業の記録作業を省力化することを目的としています。

**最大の特徴: すべての処理が端末内で完結します。音声データ・テキストデータは外部に一切送信されません。**

---

## 2. データの取り扱い

| 項目 | 内容 |
|---|---|
| 音声データの外部送信 | **なし**（端末内でのみ処理） |
| テキストデータの外部送信 | **なし** |
| クラウドサービスの利用 | **なし** |
| ユーザーアカウント・ログイン | **不要** |
| データの自動保存・収集 | **なし** |

処理に使用する AI モデル（Whisper・LLM）はすべてローカルにダウンロードして使用します。インターネット接続はモデルの初回ダウンロード時のみ必要で、以降はオフラインで動作します。

---

## 3. 使用するソフトウェア（すべてオープンソース）

| ソフトウェア | 用途 | ライセンス |
|---|---|---|
| Python 3.12 | アプリケーション実行環境 | PSF License |
| faster-whisper | 音声文字起こし（Meta Whisper の実装） | MIT |
| Ollama | ローカル LLM サーバー | MIT |
| ffmpeg | 音声ファイルの読み込み | LGPL / GPL |
| Gradio | ブラウザ UI | Apache 2.0 |
| uv | Python パッケージ管理 | MIT / Apache 2.0 |

---

## 4. インストール要件

### 必要な権限
- **初回セットアップのみ**: 管理者権限（ソフトウェアのインストール・PATH 設定）
- **通常使用時**: 管理者権限不要

### 動作環境
- OS: Windows 11
- RAM: 16GB 以上推奨
- ストレージ: 約 10GB（モデルファイル含む）
- インターネット接続: セットアップ時のみ（以降はオフライン動作）

---

## 5. ネットワーク・セキュリティ

| 項目 | 内容 |
|---|---|
| 通信先 | セットアップ時のみ（Python.org、GitHub、Hugging Face） |
| 常駐プロセス | Ollama（ローカルのみ、`localhost:11434` でリッスン） |
| 外部ポートの開放 | 不要 |
| ファイアウォール設定変更 | 不要 |

Ollama はローカルホスト（`127.0.0.1`）のみでリッスンするため、外部からのアクセスはできません。

---

## 6. セットアップの流れ

1. 管理者 PowerShell でセットアップスクリプト（`setup.ps1`）を実行（所要時間: 約 15〜30 分）
2. PC を再起動
3. 以降は `start.bat` をダブルクリックするだけで起動

---

## 7. 懸念点・制約

| 懸念事項 | 補足 |
|---|---|
| 初回のインターネット通信 | Hugging Face（米国）からモデルをダウンロードします。プロキシ環境では別途設定が必要な場合があります |
